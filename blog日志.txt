  MyBlog V1.7-9beta 总结报告
  更新日期: 2026-06-27

一、项目概述
  MyBlog 个人博客系统
  版本: V1.7-9beta
  技术栈: Node.js + Express 5 + SQLite + Cloudflare Tunnel
  端口: 3000
  启动方式:
    - 内网: blogserver.bat (node server.js)
    - 外网: blogI.bat (node start-tunnel.js + cloudflared)

二、版本 V1.7 主要更新

  1. 上传大文件修复
     - 新增 multer 错误处理中间件，上传错误返回 JSON 而非 HTML
     - 服务器超时从默认 2 分钟延长至 10 分钟
     - 前端上传增加 try-catch 和 JSON 解析容错
     - 问题: 113MB 视频上传失败 (Timeout / 无响应)
     - 涉及文件: src/routes/media.js, server.js, public/js/app.js

  2. 版本标识
     - 页脚添加版本号
     - 涉及文件: public/index.html

  3. 二级密码置顶 (V1.7-5 beta)
     - 修复二级密码弹窗被其他弹窗遮挡的问题
     - 原因: 所有弹窗 z-index 均为 200，sec-password-modal 在 DOM 中靠前，被后定义的弹窗覆盖
     - 修复: sec-password-modal 独立 z-index: 300，确保始终置顶
     - 涉及文件: public/css/main.css

  4. 大文件上传深度修复 (V1.7-7beta)
     - Express 5 的 express.json/urlencoded 可能干扰 multer 的 multipart 解析
     - 修复: 全局 body 解析器跳过 multipart 请求，直接交由 multer 处理
     - 新增: 上传请求的诊断日志 (Content-Type / Content-Length)
     - 涉及文件: server.js, src/routes/media.js

  5. 媒体关联修复 (V1.7-8beta)
     - 上传的视频/文件在存储到 post_id = NULL，新建帖子后未关联
     - 新增: PUT /api/media/link 端点，批量更新 media 的 post_id
     - 前端: savePost 新建帖子后自动调用 batchLinkMedia 关联已上传文件
     - 涉及文件: src/routes/media.js, public/js/app.js

  6. 视频识别与下载功能 (V1.7-9beta)
     - 有些浏览器上传视频时 MIME type 非 video/*，导致 filetype 被误判为 'file'
     - 修复: 新增 getFiletypeByExt() 按扩展名二次判断 (mp4/webm/mov 等)
     - 前端: isVideo/isImage 函数同时检查 filetype、MIME type、扩展名
     - 下载: 帖子详情底部所有附件都显示下载按钮
     - 涉及文件: src/routes/media.js, public/js/app.js

三、功能清单

  - 用户: 登录 / 注册 / 一级密码 / 二级密码
  - 帖子: 创建 / 编辑 / 删除 / Markdown 编辑 / 分类 / 标签 / 置顶
  - 媒体: 图片 / 视频 / 文件上传下载 / 拖拽上传
  - 展示: 暗色 / 亮色 / 赛博朋克 三种主题
  - 视频池: B站 / YouTube 外链嵌入 / 每日推荐 / 三角洲专区
  - 设置: 博客标题 / 布局排序 / 个人资料 / 备份导出
  - 统计: 数据看板 / 浏览历史 / 个人主页

四、项目结构

  server.js          - 主入口 (Express 5)
  config.js          - 全局配置
  start-tunnel.js    - Cloudflare 隧道启动
  cloudflared.exe    - Cloudflare 隧道客户端
  db/                - SQLite 数据库
  public/            - 前端静态文件
    index.html       - 主页面 (页脚版本号)
    css/             - 样式文件 (main.css / theme-*.css)
    js/              - JavaScript (app.js / auth.js / theme.js)
  src/
    routes/          - 后端路由 (auth / posts / media / categories 等)
    middleware/      - 中间件 (auth 验证)
    db.js            - 数据库初始化
  uploads/           - 上传文件存储 (images / videos / files)

五、待优化方向

  - 大文件分片上传
  - 图片压缩/缩略图
  - HTTPS 支持
  - 移动端适配优化
